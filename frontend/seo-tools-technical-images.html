Best Practices for Image Alt Text
  • Descriptive content: Alt text should describe what's in the image
  • Concise but informative: Keep it brief but descriptive (125 characters or less)
  • Include keywords: Naturally incorporate relevant keywords when appropriate
  • Context matters: Consider the image's purpose and context on the page
  • Decorative images: Use empty alt="" for purely decorative images
  • Functional images: Describe the function, not just the appearance
  • No redundancy: Don't repeat information already in surrounding text
